## Features
* User and Admin Register & Login
* Delete, update and add new product
* Order
* Responsive Design

## Tools
* ReactJS
* React Query
* Mockapi.io
* Chakra-Ui
* Ant Design
* React Bootstrap
* Context API
* Formik


## Usage

In the project directory, you can run:
```
npm start
```
